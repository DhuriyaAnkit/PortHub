# PortHub

A modern, responsive web application for showcasing candidate portfolios and connecting with hiring managers.

## Features

- **Role-Based Authentication:** Sign up and log in as either a Candidate or a Hirer.
- **Dynamic Candidate Feeds:** Browse available candidates instantly.
- **Hirer Dashboard:** Search, filter, and bookmark candidates.
- **Candidate Hub:** Edit profile bio, define up to 3 roles, link a resume, and track notifications when bookmarked.
- **Theme Switching:** Sleek Dark Mode (default) and crisp Light Mode, instantly accessible.
- **Mock LocalStorage Database:** An automated mock backend using browser memory that tracks your active sessions, candidate pool, and bookmarks.

## Getting Started

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the frontend server.
4. Open the provided `localhost` URL in your browser.

## Built With

- React (Vite)
- Standard JavaScript & CSS (No external UI libraries)
- Lucide React (Icons)
